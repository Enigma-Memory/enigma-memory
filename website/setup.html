First run

Set up your Memory Drive in one guided flow.

Enigma creates a local vault, checks supported AI clients, and shows any fix-it steps before you start using memory.

Open setup guide

What happens during first run

1

Create local vault

Enigma sets up an Enigma-controlled vault in a local app location. No cloud account or provider credential is required.

2

Connect supported apps

Enigma detects installed AI clients, previews the connection change, and applies it only after you approve.

3

Review health

The health dashboard shows whether your vault and connections are ready, and explains any fix-it actions.

4

Start using memory

Once the setup flow is green, your connected AI clients can request context from your local vault when you choose to use it.

Need command-line setup?

Use the Developer CLI appendix for npm, Node, terminal commands, and manual MCP JSON. This is the power-user path, not the default consumer setup.
