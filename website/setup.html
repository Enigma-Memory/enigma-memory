First run

Set up your Memory Drive in one guided flow.

Enigma creates a private vault on this computer, looks for supported AI apps, and shows plain fix-it steps before you start using memory.

Open setup guide

The default setup is the desktop app wizard. Command-line details stay in the advanced appendix unless support asks you to use them.

Visual path

Four screens. No terminal. No JSON.

General users should only need the desktop app: create the local drive, review detected apps, approve a connection preview, then check what happened.

What happens during first run

1

Create Memory Drive

Enigma creates a private Memory Drive on this computer. No cloud account or provider login is required.

2

Connect supported apps

Enigma finds installed AI apps, previews the connection change, and applies it only after you approve.

3

Review health

The health dashboard shows whether your vault and connections are ready, and explains any fix-it actions.

4

Start using memory

Once setup is green, your connected AI apps can ask your Memory Drive for context when you choose to use it.

Need command-line setup?

Use the Developer CLI appendix only if support asks for command-line steps or you already prefer terminal setup. It is not the default consumer path.
