About signing and security warnings

Before public launch, macOS distribution must use Developer ID signing, notarization, and stapling. Windows distribution must use a signed installer, Store, or MSIX path. Until those steps are complete, your operating system may show a security warning.

If you see a warning:

  • macOS: Right-click the installer, choose Open, and confirm you want to open it.
  • Windows: Click More info, then Run anyway, if you obtained the installer from an Enigma distribution channel.
  • Linux: Verify the package signature or checksum through your distribution tools.

Do not bypass security warnings for installers from unknown sources. See the current launch status for code-signing progress and the expected timeline.
