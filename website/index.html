How you use it

1. Check installer status

Start on the status page. If signed installers are not ready, any shared build is labeled as testing-only before you choose it.

2. Create your Memory Drive

First run creates a private Memory Drive for Enigma Memory data on this computer.

3. Connect an AI app

Choose a supported app, preview the connection, and let that app ask Enigma for memory only when you allow that use.

What Enigma does

One private Memory Drive

Enigma keeps your Memory Drive on this device. No cloud account or provider login is required for the default desktop path.

Connect your assistants

Find installed AI apps, preview what Enigma will change, and approve the connection. Claude Desktop, Cursor, Kimi Code, and more.

Health and proof

Check whether your vault and app connections are ready. Export a support-safe proof summary without exposing raw memory text.
