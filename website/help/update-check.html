Updates

How Enigma Memory checks for updates.

Update checks use the official release feed. Enigma never downloads or installs an update without your approval.

How update checks work

Enigma Memory checks the official release feed over HTTPS. The feed lists the latest version, release notes, and platform details for supported desktop builds.

Before an update is offered, Enigma checks the release information required by the app. If verification fails, the update is not offered and the app reports a verification issue in the health dashboard.

Boundary note: Update checks do not send your memory contents, prompts, wallet information, or local file paths to the update server.

Check for updates manually

  1. Open Enigma Memory.
  2. Go to Settings > General.
  3. Click Check for updates.

If an update is available, Enigma shows the version number, release notes, and an option to start the download. If no update is available, the app confirms you are on the latest version.

What "update available" means

When Enigma says an update is available, a newer release exists for your platform in the official feed. The message includes the new version number and a short summary of what changed.

Seeing "update available" does not change the app or your vault. The update is downloaded and installed only after you approve it.

No automatic downloads

Enigma does not download or install updates in the background. You choose whether to update and when.

You can also disable automatic update checks in Settings > General. When disabled, Enigma only checks for updates when you click Check for updates manually.
