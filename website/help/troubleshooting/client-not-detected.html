Troubleshooting

AI app not detected.

If you have not installed Claude, Cursor, or another supported app yet, this is not a problem. You can continue without apps and connect one later.

Symptom

Enigma Memory does not list a supported AI app during client detection, or the app status shows Not installed when you expect it to be found.

No supported apps found yet

Continue without apps. Enigma can finish setup, create your Memory Drive, and keep proof activity local before any AI app is connected.

  1. Click Continue without apps in Enigma Memory.
  2. Install Claude Desktop, Cursor, or another supported app when you are ready.
  3. Open the app once, return to Enigma Memory, then click Refresh detection.

Likely causes

  • The app was installed but never opened, so its settings area does not exist yet.
  • The app was installed in a location Enigma cannot read.
  • The app is not a supported client.

If an installed app is missing

  1. Make sure the app is installed.
  2. Open the app at least once and close it.
  3. Return to Enigma Memory and click Refresh detection.
  4. If the app appears, click Connect and approve the previewed change.

If that does not work

  1. Restart the AI app and Enigma Memory, then refresh detection again.
  2. Check that you are using a supported client version.
  3. If the app is supported but still not detected, use advanced manual settings only from the troubleshooting panel.

What to send support

Safe to share: app version, OS family and version, the connector ID shown in Enigma, and the visible support code. Do not send raw memory, prompts, transcripts, credentials, account IDs, or complete app settings files. See what is safe to share with support.

Supported app list

For the current consumer connection flow, see other supported clients or the dedicated guides for Claude Desktop and Cursor.

Advanced manual connection

Manual app settings are for developers and troubleshooting only. If the desktop app cannot detect a supported client, see the CLI appendix for manual snippets and connector commands.
