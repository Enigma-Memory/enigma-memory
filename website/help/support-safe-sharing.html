Support safety

What is safe to share with Enigma support.

Use sanitized diagnostics and support codes. Keep memory content, prompts, transcripts, credentials, account IDs, and local paths private.

Safe to share

  • The Enigma Memory app version.
  • Your operating-system family and version, such as Windows 11 or macOS 15.
  • The visible support code or sanitized error code shown by Enigma.
  • Connector ID, client name, and whether the client was detected, connected, or waiting for restart.
  • Health status labels, update status labels, and public-safe proof refs.
  • Crash report counts and the crash-reporting opt-in state.

Do not share

  • Raw memory text, imported notes, prompts, transcripts, or provider responses.
  • Credentials, access tokens, private keys, recovery phrases, wallet data, or account IDs.
  • Complete client configuration files or manual MCP JSON copied from your machine.
  • Local file paths, vault locations, folder screenshots that reveal user names, or terminal output with paths.
  • Full logs, stack traces, heap dumps, screenshots of memory contents, or screenshots of provider conversations.
Boundary note: Enigma support does not need private memory contents to troubleshoot install, connection, update, crash-reporting, or health-dashboard issues.

Before sending screenshots

  1. Prefer the in-app support code or health summary over a screenshot.
  2. If a screenshot is necessary, crop it to the Enigma status message and remove surrounding desktop, file browser, terminal, or AI chat content.
  3. Do not send screenshots that show raw memory, prompts, transcripts, account pages, or file locations.

Where this applies

  • Installer blocked: share OS family, installer channel, and the visible support code.
  • Vault not ready: share the health status and whether setup or repair was interrupted.
  • AI app not detected: share the client name, connector ID, app version, and support code.
  • Update checks: share the current version, offered version, and update status label.
  • Crash reporting: share whether crash reporting is on and how many pending reports Enigma shows.
