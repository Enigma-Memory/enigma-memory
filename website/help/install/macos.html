Install guide

Install Enigma Memory on macOS.

Download the signed, notarized installer, open it, and let the first-run wizard create your local vault. No terminal or Node install is required.

macOS install steps

  1. Download the macOS installer from the download page.
  2. Open the signed, notarized disk image or package.
  3. Drag Enigma Memory into Applications, or run the installer package.
  4. Open Enigma Memory from Applications. The first-run wizard creates your local vault.
Boundary note: The desktop app bundles its runtime. Consumers should not install Node or npm for the default path.

If macOS blocks the installer

macOS Gatekeeper may warn about apps downloaded from the web. If the installer came from an Enigma distribution channel:

  • Right-click the installer or app and choose Open.
  • Click Open in the dialog to confirm.

Do not bypass security warnings for installers from unknown sources. See the installer blocked article for more details.

Install from npm or source

This is the advanced developer path. For the default consumer experience, use the desktop app above.

If you want the command-line engine instead, see the CLI appendix for npm install, Node requirements, setup commands, and source checkout.
