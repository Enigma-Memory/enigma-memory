Install guide

Install Enigma Memory on Windows.

Download the Windows desktop installer from the official Enigma page, run it, and let the first-run wizard create your local vault. No terminal or Node install is required.

Windows install steps

  1. Download the Windows installer from the download page.
  2. Run the installer. If Windows SmartScreen appears, cancel the prompt and check the launch status. General consumers should wait for a signed build marked ready for public install.
  3. Complete the installer with the default choices.
  4. Open Enigma Memory from the Start menu. The first-run wizard creates your local vault.
Boundary note: The desktop app bundles its runtime. Consumers should not install Node or npm for the default path.

If Windows blocks the installer

Windows may show a SmartScreen warning for apps downloaded from the web. Do not continue past it by default.

  • Recommended for consumers: Cancel the prompt, check the launch status, and wait for a signed build marked ready for public install.
  • Testing-only: If you are validating a pre-launch build from an official Enigma channel, click More info, then Run anyway.

Do not bypass security warnings for unknown publishers, unofficial mirrors, email attachments, or copied links. See the installer blocked article for more details.

Install from npm or source

This is the advanced developer path. For the default consumer experience, use the desktop app above.

If you want the command-line engine instead, see the CLI appendix for npm install, Node requirements, setup commands, and source checkout.
