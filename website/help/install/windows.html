Install guide

Install Enigma Memory on Windows.

Download the signed installer, run it, and let the first-run wizard create your local vault. No terminal or Node install is required.

Windows install steps

  1. Download the Windows installer from the download page.
  2. Run the signed installer. If Windows SmartScreen appears, click More info, then Run anyway.
  3. Complete the installer with the default choices.
  4. Open Enigma Memory from the Start menu. The first-run wizard creates your local vault.
Boundary note: The desktop app bundles its runtime. Consumers should not install Node or npm for the default path.

If Windows blocks the installer

Windows may show a SmartScreen warning for apps downloaded from the web. If the installer came from an Enigma distribution channel:

  • Click More info on the SmartScreen prompt.
  • Click Run anyway.

Do not bypass security warnings for installers from unknown sources. See the installer blocked article for more details.

Install from npm or source

This is the advanced developer path. For the default consumer experience, use the desktop app above.

If you want the command-line engine instead, see the CLI appendix for npm install, Node requirements, setup commands, and source checkout.
