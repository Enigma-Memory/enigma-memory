Connect apps

Connect Enigma to supported AI clients without editing JSON.

The desktop app detects installed clients, previews the setting it will change, and applies the Enigma connection only after you approve it.

Open Connections in Enigma

Supported clients

Claude Desktop

Detected automatically. When the Enigma Desktop Extension .mcpb is available, it is the easiest Claude path.

Cursor

Detected automatically. Preview the MCP entry, approve, then restart Cursor.

Kimi Code

Detected automatically. Preview the MCP entry, approve, then restart Kimi Code.

VS Code / Cline

Detected when Cline is installed. Preview and approve the MCP settings change.

Roo Code

Detected when Roo is installed. Preview and approve the MCP settings change.

OpenCode

Detected when OpenCode is installed. Preview and approve the MCP settings change.

Generic MCP

For any other MCP client that supports stdio servers. Use the advanced panel for manual JSON.

Boundary note: Enigma connectors can only configure the client to start Enigma's local MCP server. They cannot make a provider delete logs or force a model to forget.
Manual MCP configuration

Manual MCP JSON is for developers and troubleshooting only. If the desktop app cannot detect or connect a client, see the CLI appendix for manual snippets and connector commands.
