Connect apps

Connect Claude Desktop to Enigma Memory.

The Enigma desktop app prepares the Claude Desktop Extension (.mcpb). Claude installs it from Settings → Extensions; Enigma does not write Claude settings for this extension handoff.

Connect Claude Desktop

  1. Make sure Claude Desktop is installed and has been opened at least once.
  2. Open Enigma Memory and run the setup wizard, or open the Connections screen.
  3. Choose Claude Desktop, then create the Claude Desktop Extension package.
  4. Open Claude Desktop Settings → Extensions and drag the Enigma .mcpb package in, or choose Install/Browse and select it.
  5. When Claude asks for configuration, choose your local Memory Drive file.
  6. Restart Claude Desktop after approving the extension install.
  7. Test the connection with the read-only Enigma tool enigma_support_summary or enigma_next_action.
Boundary note: Enigma's Claude extension starts Enigma's local helper from Claude Desktop. It does not write Claude settings in this extension flow, delete Anthropic logs, or force a model to forget.

Claude Desktop Extension

The .mcpb extension is the first supported Claude path. Use advanced manual settings only when support asks for a dry-run fallback.

If Claude Desktop cannot see Enigma

Try these no-write repair steps first:

  • Enable or reinstall Enigma Memory in Claude Desktop Settings → Extensions.
  • Reselect your local Memory Drive file if Claude asks for it again.
  • Fully quit and reopen Claude Desktop, then run enigma_support_summary or enigma_next_action again.

Use enigma connect claude-desktop --dry-run only as an advanced fallback when support asks. If Claude Desktop itself is not detected, see AI app not detected.

Disconnect Claude Desktop

Remove or disable Enigma Memory in Claude Desktop Settings → Extensions. This stops Claude from starting Enigma's local helper; it does not delete your Memory Drive.

Advanced manual connection

Manual app settings are for developers and troubleshooting only. If the desktop app cannot detect or connect Claude Desktop, see the CLI appendix for manual snippets and connector commands.
