Connect apps

Connect Cursor to Enigma Memory.

The Enigma desktop app detects Cursor, previews the connection change in plain language, and applies it only after you approve it.

Connect Cursor

  1. Make sure Cursor is installed and has been opened at least once.
  2. Open Enigma Memory and run the setup wizard, or open the Connections screen.
  3. When Cursor appears in the detected apps list, click Connect.
  4. Review the connection preview, then approve it.
  5. Restart Cursor if Enigma asks you to.
Boundary note: Enigma connectors can only configure Cursor to start Enigma's local helper. They cannot make Cursor or its providers delete logs or force a model to forget.

If Cursor is not detected

Try these steps:

  • Confirm Cursor is installed and was opened at least once.
  • Click Refresh detection in Enigma.
  • Restart Cursor, then refresh detection again.

If it still is not detected, see AI app not detected.

Advanced manual connection

Manual app settings are for developers and troubleshooting only. If the desktop app cannot detect or connect Cursor, see the CLI appendix for manual snippets and connector commands.
