Install guide

Install Enigma Memory like a desktop app.

Download, open, and approve the signed installer. Enigma handles the local runtime and vault setup.

Desktop install

macOS

  1. Download the macOS installer from the download page.
  2. Open the signed, notarized disk image or package.
  3. Drag Enigma Memory into Applications, or run the installer.
  4. Open Enigma Memory. The first-run setup creates your local vault.

Windows

  1. Download the Windows installer from the download page.
  2. Run the signed installer.
  3. If Windows SmartScreen appears, click More info, then Run anyway.
  4. Open Enigma Memory. The first-run setup creates your local vault.

Linux

  1. Download the AppImage or package from the download page.
  2. Mark the AppImage executable, or install the package with your distribution tools.
  3. Open Enigma Memory. The first-run setup creates your local vault.
Boundary note: The desktop app bundles its runtime. Consumers should not install Node or npm for the default path.

If your system blocks the installer

Operating systems sometimes warn about apps downloaded from the web. Use signing-safe guidance:

  • macOS Gatekeeper: Right-click the installer, choose Open, then confirm.
  • Windows SmartScreen: Click More info, then Run anyway, if the installer came from an Enigma distribution channel.
  • Linux: Check the package signature or checksum with your distribution tools.

Do not bypass security warnings for installers from unknown sources.

Install from npm or source

This is the advanced developer path. For the default consumer experience, use the desktop app above.

If you want the command-line engine instead, see the CLI appendix for npm install, Node requirements, setup commands, and source checkout.
