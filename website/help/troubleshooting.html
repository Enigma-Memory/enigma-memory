Troubleshooting

Fix common setup issues from the health dashboard.

Enigma explains what is wrong, what it can fix automatically, and what you need to approve.

Common issues

Installer blocked by the operating system

Try this first: Right-click the installer and choose Open (macOS), or click More info then Run anyway (Windows), if the installer came from an Enigma distribution channel.

What Enigma checks: Whether the installer is signed and notarized before public launch.

If that does not work: Download again from the official download page. Do not bypass warnings for installers from unknown sources.

Vault not ready

Try this first: Open Enigma and click Create local vault in the setup wizard.

What Enigma checks: The local vault bundle exists and is readable.

If that does not work: Use the in-app Repair vault option. CLI repair is available in the advanced panel.

AI app not detected

Try this first: Make sure the app is installed and has been opened at least once, then click Refresh detection in Enigma.

What Enigma checks: Whether the client config directory exists and whether Enigma can write a previewed MCP entry.

If that does not work: Restart the AI app, or use the manual MCP JSON in the advanced panel.

Proof export rejected as unsafe

Try this first: Remove any raw memory text, prompts, transcripts, or provider responses from the export fields.

What Enigma checks: The export contains only hashes, roots, opaque refs, counts, timestamps, signatures, and validation results.

If that does not work: Use the in-app proof activity screen, which exports safe artifacts automatically.

Send safe diagnostics to support

Safe to share: Health status, app version, sanitized error codes, and public-safe proof refs such as ref_abc123.

Do not share: Raw memory, prompts, transcripts, credentials, account IDs, or local paths.

CLI repair commands

If the desktop app repair options do not resolve the issue, see the CLI appendix for terminal-based repair commands. Run these only if you are comfortable with the command line.
