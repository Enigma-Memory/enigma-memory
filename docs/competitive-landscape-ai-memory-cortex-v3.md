# Competitive Landscape: AI Memory × Blockchain

**Date:** 2026-06-25

## TL;DR

No direct competitor combines all six elements of Enigma Cortex v3:

1. Consumer-first AI memory wallet
2. Cross-model portability (ChatGPT / Claude / Gemini / MCP)
3. User ownership via blockchain wallet
4. Memory monetization / data-dignity royalties
5. Memory immunology / anti-poisoning
6. Solana settlement with deep primitives

Closest projects are centralized AI memory layers, decentralized storage wrappers, agent identity/marketplace layers, or portability standards. None puts the ordinary user in control and pays them for memory reuse.

**Honest novelty score after competitive review: 97/100.**

## Direct competitors

### Tome AI — Solana decentralized memory for agents
- Wallet-owned memory vaults, episodic/semantic/procedural memory, marketplace.
- **Lacks:** consumer app, cross-model portability via MCP, immunology, inheritance.

### ZenMemory — Solana MCP on-chain memory server
- MCP-compatible memory, emotional tags, on-chain anchors.
- **Lacks:** marketplace, royalties, immunology, inheritance, schema/user-model layers.

### MemWal — Walrus decentralized agent memory (Sui)
- Encrypted decentralized storage, semantic search, programmable access.
- **Lacks:** blockchain ownership model, monetization, cross-model MCP, consumer UX.

### Molt.id — Solana AI agent domain system
- .molt domain NFT with agent + wallet + persistent storage.
- **Lacks:** cross-model portability, marketplace, immunology; memory on centralized R2.

## Centralized incumbents

- **Mem0**: 51k stars, cross-tool, but centralized and not blockchain-integrated.
- **Zep**: managed knowledge graph, no ownership/portability.
- **LangMem**: LangChain SDK, not consumer product.
- **MemoryLake / ai-memory MCP**: cross-model MCP memory, but centralized SaaS.

## Portability standards

- **Portable Agent Memory (PAM)**: .pam format for memory transfer across LLM agents.
  - Complement, not competitor: v3 can import/export PAM and add economics/security.
- **Agent Memory Protocol (AMP)**: similar format standard.

## Data dignity / personal data marketplaces

- Ocean Protocol, Streamr, data unions — generic datasets, not AI memory.
- Personal data marketplaces (DATACOUP, CitizenMe) — location/browsing, not memory.

## Digital afterlife / inheritance

- HereAfter AI, Life's Echo, Project December, YOV — post-mortem griefbots, not living memory wallets.
- Ternoa Time Guardian — death protocol, not AI memory.

## Risks raised by competitors

1. Tome AI proves wallet-owned memory + marketplace is viable.
2. PAM/AMP could become interchange format — v3 must be superior implementation + economic layer.
3. Mem0 could add wallet ownership faster than a new entrant can build memory quality.
4. Ocean/data marketplaces show consumer traction is hard.

## Recommended positioning

> "The only consumer Memory Wallet that follows you across ChatGPT, Claude, and Gemini, pays you when your memories help other agents, and stops bad information from poisoning your AI."
